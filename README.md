# Hollywood Theater

A theater ticketing website created with MERN Stack.

-   Swift
-   Texture
-   MongoDB
-   Express.js
-   Node.js

## Creating Pods in Swift

`pod init`

### In the pod file place

`pod 'Texture'`
`pod 'RxSwift'`
`pod 'ReSwift'`

## Running Backend Folder

`npm start`

## Running FrontEnd

-   Click on `MoodTracker.xcworkspace` file
